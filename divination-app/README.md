# 占卜应用 | Divination App

一个基于传统易学的现代占卜应用，提供六爻占卜、梅花易数和紫微斗数三种占卜方式。

## ✨ 功能特性

### 已实现功能

- **六爻占卜** 🪙
  - 完整仪式模式（6次独立摇卦）
  - 快速模式（一键生成）
  - 动态硬币投掷动画
  - 变卦自动计算
  - 卦辞和爻辞展示

- **梅花易数** 🌸
  - 数字起卦法（输入3个数字）
  - 时间起卦法（基于当前时间）
  - 自动计算上卦、下卦和动爻
  - 变卦展示

- **历史记录** 📜
  - 最多保存500条记录
  - 准确度标记
  - 添加备注
  - 删除和清空功能
  - 存储容量警告

- **紫微斗数** ⭐
  - 生辰八字与干支四柱配置
  - 传统天星十二宫排盘（十二宫位大限流动）
  - 命主、身主与生年四化（禄权科忌）自动计算标注
  - AI 智能命盘解读（一键复制提示词支持外部大模型二次探讨）

### 待实现功能

- **高级特性** 📈
  - 历史记录云端备份同步
  - 传统占卜知识与命理学习板块

## 🎨 设计风格

- **传统美学**：温暖的配色方案（奶油色背景 + 陶土色强调）
- **现代交互**：流畅的动画和过渡效果
- **响应式设计**：适配桌面和移动设备
- **易读性**：清晰的层次和优雅的排版

## 🛠️ 技术栈

- **框架**: React 18 + TypeScript
- **构建工具**: Vite
- **样式**: TailwindCSS
- **路由**: React Router v6
- **动画**: Framer Motion
- **存储**: LocalStorage

## 📦 安装和运行

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build

# 预览生产构建
npm run preview
```

## 🚀 部署

项目配置为部署到 Cloudflare Pages：

1. 构建命令: `npm run build`
2. 构建输出目录: `dist`
3. Node 版本: 18+

## 📖 使用说明

### 六爻占卜
1. 输入问题（可选）
2. 选择完整仪式或快速模式
3. 完成6次摇卦（或一键生成）
4. 查看本卦、变卦和解释
5. 保存到历史记录

### 梅花易数
1. 选择起卦方式（数字法或时间法）
2. 数字法：输入3个数字（前两个1-8，第三个任意正整数）
3. 时间法：直接点击开始占卜
4. 查看卦象和解释
5. 保存到历史记录

### 历史记录
1. 查看所有占卜记录
2. 展开查看详情
3. 标记准确度
4. 添加备注
5. 删除不需要的记录
