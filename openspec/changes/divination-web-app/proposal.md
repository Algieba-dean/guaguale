## Why

日常生活中常常面临各种选择困难，需要一个轻松有趣的方式来帮助思考和决策。传统卜卦方法（六爻、梅花易数、紫微斗数）虽有深厚文化底蕴，但过于严肃且不易使用。我们需要一个 Q 版传统美感的 web app，让用户在娱乐化的氛围中完成卜卦，既能体验传统文化，又能借此理清思路。

## What Changes

- 创建一个纯前端的卜卦 web 应用，可部署在 Cloudflare Pages
- 实现三种卜卦方式：六爻占卜（优先）、梅花易数、紫微斗数
- Q 版传统美感的视觉设计：暖色系配色、圆润界面、简化传统元素
- 完整的卜卦流程：从提问、起卦、到解读，支持完整展示或快速跳过
- 卦象数据库：64 卦的卦辞、爻辞、白话解读（古文+译文双语对照）
- 本地历史记录功能：保存占卜记录，支持用户标注准确度

## Capabilities

### New Capabilities
- `liuyao-divination`: 六爻占卜系统，包含摇卦交互、本卦变卦计算、卦象解读
- `meihua-divination`: 梅花易数系统，支持报数起卦和时间起卦
- `ziwei-divination`: 紫微斗数系统，基于出生时间的命理排盘
- `hexagram-data`: 64 卦数据结构和管理，包含卦辞、爻辞、解读文本
- `divination-history`: 占卜历史记录管理，本地存储和回顾功能
- `ui-components`: Q 版传统美感的 UI 组件库（铜钱动画、卦象展示等）

## Impact

- 新建完整的前端项目，使用 React + Vite + Framer Motion 技术栈
- 需要整理和创建 64 卦的 JSON 数据文件
- 纯静态部署，无后端依赖，所有数据和逻辑在前端完成
- 使用 localStorage 存储用户历史，不涉及用户账号系统
